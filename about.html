About Crossroot

Files should be easyto find.

Crossroot began with a simple frustration: a file exists somewhere, but finding out where can take longer than using it.

Our mission

Make scattered storage feel like one organized place.

People create work across computers, cloud drives, school accounts, shared folders, phones, and different companies. Crossroot is being built as the search layer above all of them.

It is not meant to become another storage service. The goal is to help people find the original file wherever it already lives.

“You should not need to remember the platform before you can remember the file.”
The idea behind Crossroot
What guides us

Designed around trust.

Search, not storage

Crossroot should help users find existing files rather than forcing them into another ecosystem.

Permission first

Users should understand exactly which accounts, folders, and file types are being indexed.

Useful before AI

Reliable filename, content, date, type, and source search comes before flashy features.

AI with clarity

When AI search arrives, Crossroot should clearly explain whether processing is local or cloud-based.

Help shape it

Crossroot is just beginning.

Early users can help decide which providers, search tools, and privacy controls matter most.
