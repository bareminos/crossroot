Information Crossroot may process

Account information, connected-service authorization data, approved file metadata, search settings, and diagnostic information may be needed to operate the service.

Local files

Crossroot should index only folders a user approves. The product should clearly disclose whether any filenames, metadata, document text, or AI inputs leave the device.

Connected services

Crossroot should use official authorization systems and request the narrowest practical permissions.

User controls

Users should be able to disconnect accounts, pause indexing, remove indexed sources, and request account deletion.
