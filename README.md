# Crossroot Website

A dark, glassy landing page prototype for Crossroot.

## Files

- `index.html` — page structure
- `styles.css` — all styling and responsive design
- `script.js` — animations, filters, mobile navigation, and demo auth modal

## Run locally

Open `index.html` directly in your browser, or use VS Code's Live Server extension.

## Important

The login form is currently a front-end demo only. Do not store real user passwords in JavaScript or GitHub. Connect it to a real authentication provider such as Supabase Auth, Firebase Auth, Clerk, or Auth0.
